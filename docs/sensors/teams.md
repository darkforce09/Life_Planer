# Microsoft Teams Integration

## Goal
To monitor Teams for group work updates, shared files, and ad-hoc lecture links.

## Data Required
- **Messages/Announcements:** Channel messages in relevant course teams.
- **Meetings:** Scheduled Teams meeting links.

## API / Access Strategy
- Microsoft Graph API.
- This might be complex depending on university Azure AD restrictions.

## Polling Frequency
- Every 2-4 hours.
